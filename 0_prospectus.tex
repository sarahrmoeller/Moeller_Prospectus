\abstract{  \OnePageChapter	% because it is very short

Machine learning can provide automated assistance to document and describe endangered languages with speed and accuracy, but linguists must know how to make effective use of it. The proposed dissertation research will focus on effective integration of machine learning into language documentation and description. It will look specifically at integration into morphological annotation and analysis. It will investigate methods for achieving optimal results and assistance from machine learning.
	}
